3. front-matter语法

layout： 布局
title: 标题
data:创建日期
updated：更新日期
tags：标签
categories：分类
author：作者
excerpt：文章摘要
layout: post # 可选：post, page, draft
sticky: 100 # 文章置顶，数字越大优先级越高
comments: true # 是否开启评论
toc: true # 是否显示目录
index_img #文章描述图片
banner_img: /images/banner.jpg # 文章顶部大图
banner_img_height: 70 # 图片高度（百分比）
banner_mask_alpha: 0.3 # 图片遮罩透明度
