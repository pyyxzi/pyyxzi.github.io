星际战甲攻略

星际战甲(Warframe)攻略

新手入门

游戏背景

星际战甲(Warframe)是一款以太阳系为背景的科幻第三人称射击游戏。玩家扮演天选者(Tenno)，操控具有超凡能力的战甲(Warframe)，与各大阵营作战。

基础操作

移动系统: 滑铲(Ctrl)、翻滚(空格)、子弹跳(Ctrl+空格)、滑行(空格长按)
战斗系统: 近战(E)、射击(鼠标左键)、瞄准(鼠标右键)、技能(1-4键)
界面导航: 星图(ESC)、装备(ESC)、MOD配置(升级)

初始选择

游戏开始时可选择三种战甲：

Excalibur(剑圣): 平衡型战甲，适合新手
Mag(磁力): 控制型战甲，有一定上手难度
Volt(伏特): 输出型战甲，灵活多变

前期发展路线

1
完成地球任务解锁金星
2
解锁火星获取更多资源
3
完成主线任务《第二个梦》
4
解锁全部星球和连接点
5
获取更多战甲和武器

战甲系统

战甲分类

输出型: Saryn, Mesa, Volt等
控制型: Vauban, Nova, Khora等
坦克型: Rhino, Inaros, Nezha等
辅助型: Trinity, Wisp, Harrow等

战甲获取

任务奖励: 部分战甲可通过完成特定任务获得
阵营声望: 通过提升特定阵营声望获取图纸
Boss掉落: 击败特定Boss获取部件图纸
市场购买: 使用白金直接购买

战甲强化

Forma(极化): 改变MOD槽极性，降低MOD消耗
Orokin反应堆: 增加MOD容量上限
Umbra Forma: 用于Umbra极性槽位

武器系统

武器类型

主武器: 步枪、狙击枪、霰弹枪、弓箭等
副武器: 手枪、投掷武器等
近战武器: 单手剑、双手剑、长柄武器、拳套等

武器获取

制作: 通过图纸和资源在铸造厂制作
Dojo研究: 在氏族Dojo研究获取图纸
任务奖励: 完成特定任务获得
市场购买: 使用白金直接购买

武器强化

Forma(极化): 改变MOD槽极性
Orokin催化剂: 增加MOD容量上限
Riven MOD: 随机属性MOD，可大幅提升武器性能

MOD系统

MOD基础

MOD是游戏中强化装备的核心系统，通过安装不同的MOD可以显著提升战甲和武器的性能。

MOD获取

敌人掉落: 不同敌人有不同的掉落表
任务奖励: 特定任务可获得稀有MOD
Void裂缝: 开启遗物获得Prime MOD
Baro Ki'Teer: 虚空商人定期出售特殊MOD

MOD升级

  • 使用Endo和Credits升级MOD
  • 满级MOD效果最强但消耗也最大
  • 可以使用Ayatan雕塑转换为Endo

重要MOD推荐

战甲: 生命力、护盾、能量效率、能量上限、技能强度
主武器: 伤害、多重射击、暴击率、暴击伤害、元素伤害
副武器: 伤害、多重射击、暴击率、暴击伤害、元素伤害
近战武器: 伤害、攻击速度、暴击率、暴击伤害、触发几率

伤害机制

伤害显示

普通伤害: 白色数字
暴击伤害: 黄色数字
对护盾暴击: 蓝色数字
过载护盾: 紫色数字
真实伤害: 红色数字

物理伤害类型

冲击伤害(Impact)

对护盾效果好，可使目标蹒跚

蹒跚效果: 可使目标蹒跚，同时使其被灭骸之刃处决阈值增加
穿刺伤害(Puncture)

对装甲效果好，可使目标虚弱

虚弱效果: 降低目标输出，增加目标受到攻击的暴击几率（加算），最高25%
切割伤害(Slash)

对血肉效果好，可造成出血

出血效果: 每秒造成基础伤害35%的真实伤害，持续6秒
切割伤害mod只能提升切割的触发占比，圣装回旋刃用切割增伤mod是无效的
如果武器无该物理属性，物理伤害mod不会为武器增加该属性

基础元素伤害

寒冷伤害(Cold)
降低目标移速，射速和攻速，效果持续6秒
第一层降低50%，后续每层加5%
第10层完全冻结敌人3秒并且取消护盾充能
提高目标受到的暴击伤害
第一层增加10%，后续每层加5%
9层加50%，对完全冻住的敌人，暴击伤害加成为100%，加算
放电伤害(Electricity)
目标和周围5米敌人受到每秒基础伤害50%的电击伤害，持续6秒，每层效果持续时间独立计算
眩晕目标，持续3秒
燃烧伤害(Heat)
降低目标50%护甲
每秒造成基础伤害50%的火焰伤害，持续6秒，每层效果持续时间独立计算
部分敌人陷入恐惧，持续4秒
毒素伤害(Toxin)
每秒造成基础伤害50%的毒素伤害（无视护盾直接对生命值造成伤害），持续6秒，每层效果持续时间独立计算

复合元素伤害

引爆伤害(Blast) = 寒冷 + 燃烧
在1.5秒后对目标及周围小范围敌人造成爆炸伤害
每层效果造成基础伤害30%，每层独立显示
最多叠加10层
在1.5秒内，叠加至最大层数或者目标死亡，立即结算所有伤害，爆炸范围增加到5米
腐蚀伤害(Corrosive) = 放电 + 毒素
降低目标护甲，持续8秒
第一层降低26%，后续每层降低6%，最多叠加10层
最大层数降低80%
携带紫石头提升至14层可降低104%，完全扒皮
毒气伤害(Gas) = 毒素 + 燃烧
毒云对目标和周围3米的敌人每秒造成基础伤害50%的毒气伤害，持续6秒，每层持续时间独立计算，无层数上限
毒气层数的提高可使毒气范围提高，每层0.3m，最高10层
磁力伤害(Magnetic) = 寒冷 + 放电
对目标护盾和超宏护盾伤害提高，持续6秒
第一层提高100%，后续每层提高25%，最高叠加至10层
使目标护盾再生失效，持续6秒
护盾打破后，每层会造成基于敌方护盾3%的电击伤害
辐射伤害(Radiation) = 燃烧 + 放电
让目标敌我不分，攻击友军，持续12秒
目标对友军造成的伤害提高
第一层100%，后续每层加50%，最高10层
玩家受到辐射伤害时，可以对友军造成伤害或者被友军伤害
病毒伤害(Viral) = 寒冷 + 毒素
提升对目标生命值造成的伤害
第一层提高100%，后续每层提高25%，最高10层

伤害计算

武器面板伤害 = (冲击 + 穿刺 + 切割) + (元素)

元素伤害组合优先级为从上到下，从左到右，武器固有元素伤害最次

异常状态触发机制

每种伤害类型都有对应的触发几率，触发几率越高，越容易造成对应的异常状态。

触发几率计算
  • 基础触发几率 = 武器对应伤害占比 × 武器触发几率
  • 实际触发几率 = 基础触发几率 × (1 + 触发几率MOD加成)
多重触发

要想在单次射击中造成多种异常状态:

  • 提高触发几率是一方面
  • 提高多重射击数也是有效手段
  • 如果武器多重射击为2.1，触发几率为100%，则每次射击可以平均造成2.1个异常状态

Smeeta Kavat: 可随机提供资源加成buff

任务攻略

主线任务

觉醒

游戏开始的引导任务，介绍基本操作和战斗系统

Vor的奖赏

第一个正式任务，击败Captain Vor，获取第一把主武器

第二个梦

揭示Tenno身份的关键任务，完成后解锁焦点系统

战甲内部

了解战甲起源的任务，揭示Orokin帝国的秘密

牺牲

揭示Ballas和Margulis故事的任务，了解Sentient的威胁

奇美拉序曲

介绍Sentient威胁的任务，为新战做准备

新战

对抗Sentient的大型任务，游戏剧情高潮

特殊任务

Archwing任务: 获取太空战甲
Railjack任务: 获取战舰
钢铁之路: 获取特殊战甲Umbra
利润收割者: 获取Protea战甲

刷取资源任务推荐

生存: 长时间生存适合刷取资源和MOD
防御: 适合刷取资源和经验
破坏: 快速完成适合刷取特定资源
捕获: 最快的任务类型，适合刷取虚空裂缝

开放世界

地球平原
赏金任务: 完成Konzu提供的赏金任务获取声望
钓鱼: 在湖泊和海洋中钓鱼获取资源
采矿: 使用采矿工具获取矿石
Eidolon猎杀: 夜晚狩猎Eidolon获取特殊资源
金星山谷
赏金任务: 完成Eudico提供的赏金任务获取声望
保护热点: 保护热点获取声望和资源
洞穴探索: 探索洞穴获取特殊资源
Orb母体猎杀: 击败Orb母体获取特殊资源
火卫二
赏金任务: 完成Mother提供的赏金任务获取声望
保存动物: 保存濒危动物获取声望
钓鱼和采矿: 获取特殊资源
Necramech: 获取和升级机甲

交易系统

交易平台
  • Maroo's Bazaar: 游戏内交易区域，位于火星
  • 氏族Dojo: 在氏族内设置交易站，最常用的交易方式
  • Warframe.market: 外部交易网站，方便查询物品价格和寻找交易对象
可交易物品
Prime部件: Prime战甲和武器部件
MOD: 稀有MOD和Riven MOD
资源: 部分特殊资源可交易
Ayatan雕塑: 可用于装饰或转换为Endo
交易技巧
  • 使用Warframe.market查询物品价格，避免被坑
  • 了解物品的稀有度和获取难度，合理定价
  • 关注虚空商人Baro Ki'Teer的商品，可能成为热门交易品
  • 囤积即将被Vault的Prime部件，等价格上涨后出售

进阶技巧

战甲搭配
Saryn + 病毒: 大范围AOE伤害，适合刷怪
Mesa + 辐射: 高单体DPS，适合击杀精英敌人
Rhino + 腐蚀: 坦克输出兼备，适合新手
Trinity + 磁力: 团队支援，适合长时间生存
武器推荐
  • 初期: Hek霰弹枪、Atomos副武器、Orthos长柄武器
  • 中期: Ignis Wraith喷火器、Pyrana Prime副武器、Lesion长柄武器
  • 后期: Kuva Bramma弓箭、Kuva Nukor副武器、Kronen Prime双刀
刷取经验技巧
Hydron(土星): 经典刷经验地点
ESO(精英圣殿Onslaught): 高效刷战甲经验
SO(圣殿Onslaught): 适合刷武器经验
Adaro(塞达娜): 隐身刷近战经验
提升效率
  • 使用正确的元素组合针对敌人弱点
  • 学习快速移动技巧(子弹跳、滑铲连招)
  • 合理使用Forma和催化剂/反应堆
  • 加入活跃氏族获取更多资源和支持
